Student Management System (File Handling)

Description:
This project is a console-based Java application that stores student
records permanently using a text file.

Files:
1. Student.java
2. FileManager.java
3. StudentApp.java
4. students.txt

Features:
- Add Student
- View Students
- Search Student
- Permanent File Storage

Technologies:
- Java
- FileWriter
- FileReader
- BufferedReader
- ArrayList

Data Format:
Each record is stored as:

ID,Name,Age

Example:
101,Rahul,20
102,Priya,21

How to Run:
1. Compile:
   javac *.java

2. Run:
   java StudentApp
